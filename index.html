WEEK 04

Mastering Loops

Loops help us repeat tasks efficiently. Learn for, while and Python’s take on do-while.

1️⃣ For Loop

A for loop is used when you know how many times a loop should run.

It iterates over sequences like lists, strings, or ranges.

1
for i in range(1, 6):
2
print(i)

✅ Prints numbers from 1 to 5

2️⃣ While Loop

A while loop runs as long as a condition remains True.

⚠️ Be careful — wrong condition can cause an infinite loop!
1
count = 1
2
while count <= 5:
3
print(count)
4
count += 1

3️⃣ Do-While Loop (Python Style)

Python does not have a native do-while loop.

We simulate it using while True.

1
while True:
2
print("Runs at least once")
3
break
